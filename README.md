# 프로젝트 안내

이 문서는 **폴더와 파일이 각각 무엇을 하는지**, 그리고 **컴퓨터 터미널(검은 창)에서 어떻게 실행하는지**만 간단히 설명합니다.

---

## 1. 폴더 구조와 각 부분의 역할

프로젝트는 마치 **도서관 + 질문 답변 창구**를 만드는 작업과 비슷합니다. 문서를 넣고(`data`), 규칙을 정하고(`prompts`), 실제로 답을 만드는 코드(`src`)가 나뉘어 있습니다.

| 위치 | 쉬운 설명 |
|------|-----------|
| **`src/`** | 프로그램의 **본체**가 모인 폴더입니다. 질문을 받아 답을 만들고, 나중에 문서 검색·인덱스 같은 기능도 여기에 붙일 예정입니다. |
| `src/config.py` | API 키·모델 이름 같은 **설정**을 읽어 옵니다. |
| `src/generate.py` | OpenAI 챗 모델을 불러 **실제 답변 문장**을 만듭니다. 대화 기록도 잠깐 기억해 둡니다. |
| `src/pipeline.py` | 터미널에서 **질문을 입력받고 답을 출력**하는 **대화형 화면** 역할을 합니다. |
| `src/ingestion.py` | (준비 중) 나중에 **원본 문서를 읽어서** 처리할 자리입니다. |
| `src/embeddings.py` | (준비 중) 글을 **숫자 벡터**로 바꿔 검색에 쓰기 위한 자리입니다. |
| `src/vector_store.py` | (준비 중) 벡터를 **저장하고 비슷한 글을 찾는** 자리입니다. |
| `src/retrieve.py` | (준비 중) 질문과 관련된 **문서 조각을 찾아오는** 자리입니다. |
| **`prompts/`** | AI에게 **어떤 말투·규칙으로 답할지** 적어 두는 글(프롬프트)이 들어갑니다. |
| `prompts/system_prompt.txt` | 챗봇의 **기본 행동 지침** 텍스트입니다. |
| **`data/raw/`** | 사람이 넣는 **원본 자료**(PDF 등)를 두는 곳입니다. |
| **`data/processed/`** | 프로그램이 만든 **가공·인덱스 결과**를 두는 곳입니다. |
| **`scripts/`** | 한 번씩 돌리는 **작업용 스크립트**가 들어갑니다. |
| `scripts/build_index.py` | (준비 중) 문서로부터 **검색용 인덱스를 만드는** 실행 파일 예정입니다. |
| **`api/`** | (준비 중) 웹이나 다른 서비스에서 호출할 **API 서버**를 둘 자리입니다. |
| `api/server.py` | API 서버 코드가 들어갈 파일입니다. |
| **`main.py`** | (준비 중) 앱 전체의 **시작점**으로 쓸 수 있는 파일입니다. 지금은 비어 있습니다. |
| **`requirements.txt`** | 이 프로젝트에 필요한 **파이썬 부품 목록**(라이브러리)입니다. |
| **`.env`** | API 키처럼 **비밀 정보**를 적는 파일입니다. Git에는 올리지 않습니다. |
| **`.gitignore`** | Git이 **무시할 파일**(비밀, 캐시 등)을 정해 둔 목록입니다. |

## 2. CLI(터미널)에서 모델 실행하는 방법

**CLI**는 마우스로 클릭하는 화면이 아니라, **글자로 명령을 입력하는 검은 창**(Windows에서는 “명령 프롬프트”, “PowerShell”, “터미널” 등)을 말합니다.

### 한 번만 해두면 되는 준비

1. 이 프로젝트 폴더(`prototype_bot`)로 이동합니다.
2. Python 가상환경을 만들고 켭니다 (다른 프로젝트와 부품이 섞이지 않게 하려고 합니다).

   ```powershell
   cd c:\MinHyeok\prototype\prototype_bot
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```

3. 필요한 라이브러리를 설치합니다.

   ```powershell
   pip install -r requirements.txt
   ```

4. 폴더 안에 **`.env`** 파일을 두고, OpenAI API 키가 들어가 있어야 합니다. (`OPENAI_API_KEY=...` 형식. 키는 타인에게 공유하지 마세요.)

### 매번 대화할 때 (실행)

프로젝트 폴더에서 가상환경이 켜진 상태로 아래를 입력합니다.

```powershell
python -m src.pipeline
```

- 화면에 **`[질문]`** 이 나오면, 키보드로 질문을 입력하고 Enter를 누릅니다.
- **`[답변]`** 아래에 AI가 만든 답이 출력됩니다.
- 질문 칸에서 **아무 것도 입력하지 않고 Enter**만 누르면 **종료**됩니다.

> **참고:** 실행할 때는 반드시 **`prototype_bot` 폴더를 기준**으로 명령을 실행해야 합니다. 그래야 `prompts/system_prompt.txt` 같은 경로가 올바르게 찾아집니다.
